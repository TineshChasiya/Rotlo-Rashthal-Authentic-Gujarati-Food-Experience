# 🍽️ Rotlo Rashthal – Authentic Gujarati Food Experience

A modern and responsive restaurant website that showcases traditional Gujarati cuisine with a clean and user-friendly design.

## 🚀 Live Demo
🔗https://rotlo-rashthal.netlify.app/

## 📌 About the Project
Rotlo Rashthal is a frontend website inspired by authentic Gujarati food culture. It presents popular dishes like Gujarati thali, farsan, and sweets with an attractive layout and smooth user experience.

The goal of this project is to combine traditional food presentation with modern web design.

## ✨ Features
- Clean and modern UI design  
- Fully responsive (mobile-friendly)  
- Smooth navigation  
- Attractive food sections (menu, gallery, contact)  
- Cultural color theme inspired by Gujarati cuisine  

## 🛠️ Technologies Used
- HTML5  
- CSS3  

## 📂 Sections Included
- Home  
- About  
- Menu  
- Gallery  
- Contact  

## 💡 Purpose
This project is built to practice frontend development and create a visually appealing restaurant website design.
